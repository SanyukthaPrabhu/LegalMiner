# ⚖️ LegalMiner: Bilingual AI Legal Auditor
**VTU 6th Sem AIML Project** Developed by: Sanyuktha Prabhu

## 🚀 Features
- **Bilingual OCR:** Extracts text from English and Kannada legal documents.
- **RAG Architecture:** Queries a local ChromaDB containing the Indian Contract Act.
- **AI Auditing:** Uses Google Gemini to identify legal risks and unfair clauses.

## 🛠️ Tech Stack
- Python, Streamlit, LangChain, EasyOCR, Google Gemini API, ChromaDB.
